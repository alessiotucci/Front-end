### 1. **Personal Portfolio Page (Common Tags)**
- **HTML focus**: `<html>`, `<head>`, `<body>`, `<h1>`â€“`<h6>`, `<p>`, `<a>`, `<img>`, `<ul>/<ol>/<li>`, `<div>`, `<span>`
- **CSS focus**: colors, fonts, margins, padding, borders, background, `display: block/inline`
- **Goal**: Create a simple one-page portfolio with sections like "About Me," "Projects," and "Contact."

### 2. **Responsive Blog Layout**
- **HTML focus**: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<footer>`
- **CSS focus**: Flexbox, media queries, responsive images (`max-width: 100%`)
- **Goal**: Build a blog with a navigation bar, multiple articles, and a footer that adapts to mobile screens.

### 3. **Interactive Form Page**
- **HTML focus**: `<form>`, `<input>` (text, email, password, checkbox, radio), `<label>`, `<textarea>`, `<select>/<option>`, `<button>`
- **CSS focus**: pseudo-classes (`:hover`, `:focus`, `:checked`), form styling, transitions
- **Goal**: Create a styled registration form with validation hints and hover effects.

### 4. **Image Gallery with CSS Grid**
- **HTML focus**: `<figure>`, `<figcaption>`, `<picture>`, `<source>`
- **CSS focus**: CSS Grid, object-fit, filters (`filter: grayscale()`), `position` (absolute/relative)
- **Goal**: Build a responsive image gallery with captions and optional filters.

### 5. **Multimedia Landing Page**
- **HTML focus**: `<video>`, `<audio>`, `<source>`, `<iframe>`
- **CSS focus**: animations (`@keyframes`), transforms (`rotate`, `scale`), z-index layering
- **Goal**: Create a landing page with embedded video, audio player, and animated elements.

### 6. **Accessible Resume Page**
- **HTML focus**: `<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<caption>`, `<abbr>`, `<address>`, `<time>`
- **CSS focus**: table styling, pseudo-elements (`::before`, `::after`), custom properties (`--variables`)
- **Goal**: Build a resume with a skills table, semantic tags for accessibility, and styled pseudo-elements.

### 7. **Advanced Web Playground**
- **HTML focus**: `<canvas>`, `<svg>`, `<details>/<summary>`, `<meter>`, `<progress>`, `<datalist>`
- **CSS focus**: advanced selectors (`:nth-child`, `:not`), clip-path, gradients, CSS variables, custom animations
- **Goal**: Create a playground page with a responsive canvas drawing, SVG icons, collapsible sections, and progress bars.
